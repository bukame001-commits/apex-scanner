Partial High Quality Text

Here's how to improve the index. All of the files have been scanned and OCRed (optical character recognition) in order to turn the hardcopy texts into searchable text.

However, OCR text has many mistakes by default. To correct this a human needs to correct the files with a text editor. This directory has some files that have been partially corrected which will be useful for creating the complete correct files.

The text/hi-quality directory has some completed corrected files. The final goal will be to create HTML files which go in text/html. The html files have html line breaks in the right places and have <h1> and <h2> tags around the section headers so that search engines know which parts of the file are topic headings.

The text/ocr directory contains all the raw OCR text files that need to be manually corrected. Having the corrected files will also make the keyword search much more accurate.

If you'd like to contribute your corrected texts to this index, please use the contact form at http://cosmicchannelings.com and it will be included in the next edition of the index.
